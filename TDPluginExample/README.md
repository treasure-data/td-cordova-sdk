# TD Mobile SDK - Cordova Demo App

## Installation
### Android
Add plugin
```
$ cordova plugin add --link ..
```

Remove plugin
```
$ cordova plugin remove com.treasuredata.cordova
```

Dialog plugin

```
$ cordova plugin add cordova-plugin-dialogs
```

Build

```
$ cordova build android
```

Run with emulator

```
$ cordova emulate android
```
